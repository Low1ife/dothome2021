HTML 속성 검색하기

  • a 태그는 다른 페이지 이동을 설정합니다.
  • abbr 태그는 줄임말이나 머리글자를 표현하는데 사용합니다.
  • acronym 태그는 줄임말이나 머리글자를 표현하는데 사용합니다. 현재는 사용하지 않습니다.
  • address 태그는 웹 페이지의 연락처 정보를 설정합니다.
  • applet 태그는 애플랫 요소를 설정합니다. 현재는 사용하지 않습니다.
  • area 태그는 이미지 맵의 영역을 정의합니다.
  • article 태그는 사이트의 독립적인 컨텐츠 섹션을 설정합니다.
  • aside 태그는 주요 콘텐츠의 보조적 컨텐츠 섹션을 설정합니다.
  • audio 태그는 오디오 파일을 설정합니다.
  • b 태그는 다른 텍스트와 구별 할 때 설정합니다.
  • base 태그는 모든 문서의 기본이 되는 URL을 설정합니다
  • basefont 태그는 문서의 기본 폰트, 사이즈, 종류를 설정합니다. 지금은 CSS에서 이 기능을 구현하기 때문에 현재는 사용하지 않습니다.
  • bdi 태그는 텍스트의 출력 방향 영역을 설정합니다.
  • bdo 태그는 텍스트의 방향을 설정합니다. 텍스트의 읽는 방향이 다른 나른 언어를 위해 사용합니다.
  • big 태그는 텍스트의 크기를 크게 설정합니다. CSS가 그 기능을 대신하기 때문에 현재는 사용하지 않는 태그입니다.
  • blockquote 태그는 긴 문장의 인용문을 설정합니다.
  • blink 태그는 텍스트를 깜박이게 설정합니다.
  • body 태그는 해당 HTML 문서의 텍스트, 하이퍼링크, 이미지, 리스트 등과 같은 모든 콘텐츠를 포함하는 영역을 정의할 때 사용합니다.
  • br 태그는 텍스트 줄바꿈이 필요할 때 설정합니다.
  • button 태그는 클릭 할 수 있는 버튼을 설정합니다.
  • canvas 태그는 비트맵 방식의 그림을 설정합니다
  • caption 태그는 표의 제목을 설정합니다.
  • center 태그는 텍스트를 가운데 정렬합니다. CSS 속성이 기능을 대신하고 있기 때문에 현재는 사용하지 않습니다.
  • cite 태그는 저작물의 출처 표기를 설정합니다.
  • code 태그는 코드를 나타내는 텍스트를 설정합니다.
  • col 태그는 colgroup 요소에 속하는 각 열의 속성을 설정합니다.
  • colgroup 태그는 표의 열을 묶는 그룹을 설정합니다.
  • data 태그는 주어진 요소의 데이터 값을 설정합니다.
  • datalist 태그는 선택 목록과 자동 완성 기능을 설정합니다.
  • dd 태그는 설명 목록의 항목을 설정합니다.
  • del 태그는 문서의 삭제된 텍스트를 설정합니다.
  • details 태그는 접기/펼치기 목록을 설정합니다. 컨텐츠 요소를 숨기거나 보여줄 수 있습니다.
  • dfn 태그는 용어를 정의할 때 설정합니다.
  • dialog 태그는 팝업 정보을 설정합니다.
  • dir 태그는 파일 및 폴더의 디렉토리를 설정합니다. 현재는 사용하지 않습니다.
  • div 태그는 문서의 영역을 나누거나 그룹화 할 때 설정합니다.
  • dl 태그는 설명 목록을 설정합니다.
  • dt 태그는 설명 목록 항목의 제목을 설정합니다.
  • em 태그는 텍스트를 강조할 때 설정합니다.
  • embed 태그는 외부 콘텐츠를 사용할 수 있도록 설정합니다.
  • fieldset 태그는 폼 컨트롤 요소의 내부 영역을 설정합니다.
  • figcaption 태그는 figure 요소의 설명 또는 제목을 설정합니다.
  • figure 태그는 인용, 도표, 사진, 코드 등의 독립적인 콘텐츠를 설정합니다.
  • font 태그는 폰트의 크기, 색, 종류를 설정합니다.
  • footer 태그는 사이트 관련 정보 및 관련 링크 콘텐츠 섹션을 설정합니다.
  • form 태그는 웹 서버에 정보를 제공하는 폼 컨트롤 유형을 설정합니다.
  • frame 태그는 다른 HTML문서가 표시 될 수 있는 특정 영역을 설정합니다.
  • frameset 태그는 레이아웃 구성을 위한 프레임을 설정합니다.
  • h1~h6 태그는 섹션의 제목 설정합니다.
  • head 태그는 제목과 스크립트, 스타일 시트, 메타 정보 등을 제공합니다.
  • header 태그는 사이트의 소개 및 메뉴 그룹을 나타내는 섹션을 설정합니다.
  • hgroup 태그는 제목 요소 그룹화를 설정합니다.
  • hr 태그는 단란 요소간의 주제별 구분을 설정합니다.
  • html 태그는 웹 문서의 최상위 요소 문서의 루트를 나타냅니다.
  • i 태그는 특별한 이유로 평범한 글자와 구분하기 위해 설정합니다.
  • iframe 태그는 웹 문서에 다른 문서를 포함하는데 사용합니다.
  • img 태그는 웹 문서에 이미지를 정의합니다.
  • input 태그는 데이터를 입력할 수 있는 폼 컨트롤을 정의합니다.
  • ins 태그는 문서에 삽입된 텍스트를 설정합니다
  • kbd 태그는 키보드 입력요소를 나타내는 텍스트를 정의합니다.
  • keygen 태그는 보안키를 생성하고 컨트롤합니다.
  • label 태그는 해당 input 요소의 제목을 설정합니다.
  • legend 태그는 fieldset 요소 영역의 설명을 설정합니다.
  • li 태그는 목록의 항목을 나타내는 태그입니다.
  • link 태그는 외부 파일 연결 할 때 설정합니다.
  • main 태그는 문서의 주요 콘텐츠를 정의합니다.
  • map 태그는 이미지 맵을 정의합니다.
  • mark 태그는 참조용 표시를 정의합니다.
  • marquee 태그는 텍스트의 스크롤과 움직임을 정의합니다.
  • menu 태그는 메뉴 리스트를 정의합니다.
  • menuitem 태그는 메뉴 리스트 항목을 정의합니다.
  • meta 태그는 웹 문서에 대한 정보를 제공합니다.
  • meter 태그는 범위를 측정하는 게이지 정보를 정의합니다.
  • nav 태그는 내부 페이지 및 외부 페이지에 대한 탐색 링크를 제공하는 페이지 섹션을 설정합니다.
  • noframes 태그는 프레임이 지원되지 않을 경우 대체 메세지를 표시합니다.
  • noscript 태그는 스크립트를 지원하지 않는 브라우저를 위한 대체 태그입니다.
  • object 태그는 웹 문서에 포함된 멀티미디어 요소를 정의합니다.
  • ol 태그는 순서가 있는 목록형 태그입니다.
  • optgroup 태그는 select 메뉴 option 리스트의 그룹화를 설정합니다
  • option 태그는 select 요소의 항목을 정의합니다.
  • output 태그는 사용자에 컨트롤에 따라 결과값을 계산으로 보여주는 요소입니다.
  • p 태그는 단락을 나타냅니다. 문단은 하나로 묶을 수 있는 짧막한 단위를 말하며, 단락은 하나 하나의 짧은 토막이라고 말 할 수 있습니다.
  • param 태그는 object 태그의 매개변수를 정의합니다.
  • picture 태그는 화면 크기에 따라 이미지를 정의합니다.
  • pre 태그는 마크업 소스를 화면 그대로 표현합니다.
  • progress 태그는 작업의 진행 상태를 표시하는 바를 나타냅니다.
  • q 태그는 짧은 글을 인용할 때 사용합니다.
  • rp 태그는 루비 태그가 지원되지 않을 경우 괄호('(')를 표현해주는 역할을 합니다.
  • rt 태그는 상단에 작은 주석 글씨를 나타냅니다.
  • rtc 태그는 텍스트 컨테이너 역할을 합니다.
  • ruby 태그는 루비 문자를 정의합니다.
  • s 요소는 텍스트에 취소선을 나타냅니다
  • samp 태그는 컴퓨터 프로그램으로 출력된 결과물을 나타내는 텍스트를 표현합니다.
  • script 태그는 실행 가능한 스크립트 코드를 포함하거나 외부 스크립트를 연결할 수 있습니다.
  • section 태그는 주제별 그룹의 콘텐츠 섹션을 나타낼 때 사용합니다.
  • select 태그는 선택 목록을 설정합니다.
  • small 태그는 저작권 및 법적인 텍스트에 설정합니다.
  • source 태그는 picture, audio 요소 및 video 요소에 소스를 지정하는데 사용합니다. 닫는 태그가 없는 빈요소입니다.
  • span 태그는 콘텐츠 요소를 표현하기 위한 인라인 요소이며, 의미가 없는 그룹화 요소입니다.
  • strike 태그는 현재 사용하지 않는 태그입니다. 텍스트 위에 취소선(수평선)을 표현하기 위해 사용한 태그이지만 현재는 del 태그 또는 s 태그가 그 역할을 대신하고 있습니다.
  • strong 태그는 텍스트의 중요성을 강조할 때 설정합니다.
  • style 태그는 웹 문서에 스타일 정보를 설정할 수 있습니다.
  • sub 태그는 텍스트의 아래첨자를 사용할 때 쓰입니다.
  • summary 태그는 접기/펼치기 목록의 제목을 summary 태그를 통해 설정합니다.
  • sup 태그는 텍스트의 위첨자를 사용할 때 쓰입니다.
  • SVG 는 텍스트 기반의 표준 방식이며, HTML이나 CSS를 이용하여 사용 할 수 있습니다.
  • table 태그는 표를 설정합니다.
  • tbody 태그는 표의 본문 영역을 그룹화합니다.
  • td 태그는 표의 열(셀)을 설정합니다.
  • template 태그는 페이지가 로드 될 때 렌더링되지 않고 자바스크립트를 이용하여 HTML을 표현해주는 태그입니다.
  • textarea 태그는 여러줄의 텍스트를 편집할 수 있습니다.
  • tfoot 태그는 표의 푸터 영역을 그룹화합니다.
  • th 태그는 행이나 열에 타이틀을 설정합니다.
  • thead 태그는 표의 헤더 영역을 그룹화합니다.
  • time 태그는 시간의 특정 기간을 나타낼 때 사용합니다.
  • title 태그는 브라우저의 제목 또는 탭에 표시되는 문서의 제목을 표시합니다.
  • tr 태그는 표의 행을 설정합니다.
  • track 태그는 video 태그와 audio 태그의 자막을 설정합니다.
  • tt 태그는 타자모양의 텍스트를 보여주기 위한 태그입니다.
  • u 태그는 HTML4에서는 밑줄을 표시하기 위해서 사용하였습니다.
  • ul 태그는 일반적인 글 보다는 목록 형식의 글을 사용할 때 씁니다.
  • var 태그는 수학적인 변수나 프로그램의 변수를 쓸 때 사용합니다.
  • video 웹 문서에 동영상을 재생하는 미디어 플레이어입니다.
  • wbr 태그는 줄바꿈 할 위치를 정의할 때 사용합니다.
