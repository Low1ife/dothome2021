01. 속성 메서드 : addClass() / removeClass()

클릭하면 모든 이미지에 클래스를 추가합니다. 클릭하면 모든 이미지에 클래스를 삭제합니다. 
 {
    $(".btn1-1").click(function(){
        $(".ex1 div").addClass("active");
    });
    $(".btn1-2").click(function(){
        $(".ex1 div").removeClass("active");
    });
}

02. 속성 메서드 : toggleClass()

이미지를 클릭하면 모든 이미지에 클래스를 추가 / 삭제 합니다. 
 {
    // 모든 이미지 선택
    $(".ex2 div").click(function(){
        $(".ex2 div").toggleClass("active");
    });
    // 클릭한 이미지만 선택
    $(".ex2 div").click(function(){
        $(this).toggleClass("active");
    });
}

03. 속성 메서드 : hasClass()

리셋 클릭하면 active클래스를 찾은후 클래스를 제거합니다. 
 {
    //리셋버튼을 클릭하면 첫번째 네번째에 클래스를 추가
    $(".btn3-0").click(function(){
        $(".ex3 > div:nth-child(3)").addClass("active");
        $(".ex3 > div:nth-child(4)").addClass("active");
    });

    $(".btn3-1").click(function(){
        if($(".ex3 > div").hasClass("active")){
            $(".ex3 > div.active").removeClass("active");
        }
    });
}

04. 속성 메서드 : width(), innerwidth() , outerwidth()

여기는 텍스트 영역입니다.

클릭하면 원의 가로(width) 값을 구합니다. 클릭하면 원의 가로(innerwidth) 값을 구합니다. 클릭하면 원의 가로(outerwidth) 값을 구합니다. 클릭하면 원의 가로(outerwidth(true)) 값을 구합니다. 클릭하면 원의 세로(height) 값을 구합니다. 클릭하면 원의 세로(innerHeight) 값을 구합니다. 클릭하면 원의 세로(outerHeight) 값을 구합니다. 클릭하면 원의 세로(outerHeight(true)) 값을 구합니다.
클릭하면 샘플 박스의 가로 /세로 값을 구합니다. 클릭하면 문서의 가로 /세로 값을 구합니다. 클릭하면 브라우저의 가로 /세로 값을 구합니다. 
 {
    function demension(name , size){
        $(".text4").text("원의" + name + "값은" + size + "px 입니다." );
    }


    $(".btn4-1").click(function(){
        demension("가로(width)" , $(".circle4").width());   
    });

    $(".btn4-2").click(function(){
        demension("가로(innerwidth)" , $(".circle4").innerWidth());
    });

    $(".btn4-3").click(function(){
        demension("가로(outerwidth)" , $(".circle4").outerWidth());
    });

    $(".btn4-4").click(function(){
        demension("세로(height)" , $(".circle4").height());
    });

    $(".btn4-5").click(function(){
        demension("세로(innerheight)" , $(".circle4").innerHeight());
    });

    $(".btn4-6").click(function(){
        demension("세로(outerheight)" , $(".circle4").outerHeight());
    });

    $(".btn4-7").click(function(){
        $(".text4").text("샘플 박스의 가로 값은" + $("#sample04").width() + "px입니다.")
    });

    $(".btn4-8").click(function(){
        $(".text4").text("문서의 가로 값은" + $(document).width() + "px입니다.")
    });

    $(".btn4-9").click(function(){
        $(".text4").text("브라우저의 가로 값은" + $(window).width() + "px입니다.")
    });
}

05. 속성 메서드 : .offset()/ .position()

리셋 클릭하면 원의 가로 값을 300px로 변경합니다. 클릭하면 원의 세로 값을 300px로 변경합니다. 클릭하면 원의 가로 세로 값을 300px로 변경합니다. 
 {

06. offset() / position()

여기는 텍스트 영역입니다.

클릭하면 원의 x축 위치(offset) 값을 구합니다. 클릭하면 원의 y축 위치(offset) 값을 구합니다. 클릭하면 원의 x축 위치(position) 값을 구합니다. 클릭하면 원의 y축 위치(position) 값을 구합니다. 
 {
    $(".btn6-1").click(function(){
        $(".text6").text("이 원의 x축 값은" + parseInt($(".circle6").offset().left) + "px입니다.");
    });

    $(".btn6-2").click(function(){
        $(".text6").text("이 원의 y축 값은" + parseInt($(".circle6").offset().top) + "px입니다.");
    });

    $(".btn6-3").click(function(){
        $(".text6").text("이 원의 x축 값은" + parseInt($(".circle6").position().left) + "px입니다.");
    });

    $(".btn6-4").click(function(){
        $(".text6").text("이 원의 y축 값은" + parseInt($(".circle6").position().top) + "px입니다.");
    });
}
